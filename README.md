# Python To-Do List Application

# Project Overview
The **Python To-Do List Application** is a command-line based project developed using core Python concepts.  
It helps users manage their daily tasks by allowing them to **add, view, update, and delete tasks** through a simple menu-driven interface.

This project does not use any external libraries, files, or databases, making it lightweight and beginner-friendly.

---

# Objective
The main objectives of this project are:
- To understand and implement **basic Python programming concepts**
- To practice **logical thinking and control flow**
- To build a **real-world task management application**
- To provide a simple and interactive user experience through the terminal

---

# Features
- Add new tasks
- View all tasks with numbering
- Update existing tasks
- Delete tasks
- Menu-driven interface
- Input validation for better user experience

---

# Technologies Used
- **Programming Language:** Python  
- **Interface:** Command Line Interface (CLI)  
- **Data Storage:** In-memory list (temporary)

---

# Concepts Implemented
- Lists
- Loops (`while`)
- Conditional statements (`if-elif-else`)
- Functions
- User input handling
- Index validation
- Basic error handling
